Walkthrough

The walkthrough feature shows you around the application automatically. You can start the walkthrough by either clicking on the "View the Walkthrough" button on the beginning search page, or by clicking on the ? symbol in the bottom left of the application. The symbol will start the walkthrough from whichever page you are on currently. Here are a few points to help you navigate the Walkthrough:

  • "Exit" stops the Walkthrough.
  • "Continue" forwards the Walkthrough to the next page.
  • "Next" advances the Walkthrough to the next information item on a page.
  • "Back" returns the Walkthrough to the previous information item on a page.
  • "Okay" temporarily stops the Walkthrough feature for a required entry (main page only), then continues with the Walkthrough.

Reset Buttons

There are multiple reset buttons throughout the application. The reset button on the navigation sidebar will reset all changes made to the application. The reset buttons that show up next to the titles of each page will reset all the data on that specific page. The Customize - Quality of Life and Services pages both have reset buttons inside the cards. These will only reset the specific card you are looking at.

Customize Data

It is recognized that county-level data does not always reflect conditions in a community. The Customize Data feature allows you to change metric-level information by entering your own data values, one metric at a time, to adjust measures displayed in the Community Snapshot dashboard so indicators more closely reflect local conditions.

Load Custom Service Metrics

The Scenario Builder page has a "Load Custom Service Metrics" button next to its reset button. This will load indicator data you previously customized, using the Customize Data feature, for use in the Scenario Builder. Normally, these pages can be customized separately.

File / settings menu

The File menu contains the options to open and save community data. The save feature allows you to save your customizations as a .JSON formatted file. The load feature lets you load in a previously saved .JSON file.

The Settings menu contains the options to toggle the community search method or turn on the DISC description shown at the start-up. The search method option will change the text-based search bars to drop down menus. The "Show DISC description" option will re-enable the popup that appears when opening the application for the first time, if the "don't show this message again" checkbox was selected. You will need to restart the application after clicking this option in order to see the popup again.
